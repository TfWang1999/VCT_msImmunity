main_virtual_trials.m
This script executes in silico clinical trials for 1,000 virtual patients across four therapeutic regimens: Xelox, Xelox+IR, Xelox+Anti-PD1, and Triple Therapy. Utilizes MATLAB Parallel Computing Toolbox for high-throughput simulation.

main_quantify.m
This script processes results from Virtual Clinical Trials (VCT) to evaluate the predictive power of three immunity metrics: msImmunity (multiscale), tsImmunity (tissue-scale), and molsImmunity (molecular-scale).
